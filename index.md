## 語錄發送指令B類
>[!Note] 指令
> - **/yuzusoft 【game】【character】【quote】[enable_spoiler]**

>[!Important] 說明
> - **【game】：柚子社五大作之遊戲（必選選項）**
> - **【character】：選擇的遊戲所有的角色（必選選項）**
> - **【quote】：依據所選的角色之所有語錄（必選選項）**
> - **[enable_spoiler]：是否開啟暴雷模式（可選選項，預設False）**
>
>其中暴雷模式與A類語錄指令相同。
![image](https://hackmd.io/_uploads/r1vjorW5ge.png)
![image](https://hackmd.io/_uploads/rJ3ioBZ9xg.png)

>[!Warning]與A類語錄指令的差別
可以減少搜尋特定語錄的時間（因為Discord規定指令清單最多顯示25個選項，但本機器人每個遊戲語錄隨便都超過50個）

>[!Caution] 機器人語錄清單
>#### [點我前往查看語錄清單](https://hackmd.io/@yukicon0321/yuzusofttos#%E6%A9%9F%E5%99%A8%E4%BA%BA%E8%AA%9E%E9%8C%84%E6%B8%85%E5%96%AE)

## 申請語錄指令
>[!Note]指令列表
>- **/語錄申請 【game】【character】【quote_text】【image】[note]**


>[!Tip]申請語錄提交
>#### 參數說明
> - 【game】：必選選項，選擇要申請語錄截圖的遊戲。
> - 【character】：必選選項，填寫要申請語錄截圖的角色名稱（以【名字】內為主，例如遊戲內顯示【芳乃】填寫「芳乃」）。
> - 【quote_text】：必選選項，填寫要申請語錄截圖之該角色說的話或內心在思考的話（標點符號大部分以全形為主，例如` ， `、` 、 `、` 。 `、` ！ `、` ？ `。` ...... `則以半形` . `為主，有`「」`標點符號不用加進去）。
> - 【image】：必填選項，上傳語錄截圖檔案。
> - [note]：非必選選項，可以打備註的地方。
> ![image](https://hackmd.io/_uploads/HkIExVBreg.png)

> :::

## 抓柚子廚槍斃/抓ㄌㄌㄎ砲決管理設定
>[!Note]指令列表
>- **/抓柚子廚槍斃管理設定**
>- **/抓ㄌㄌㄎ砲決管理設定**

>[!Important] 說明
>可以調整伺服器關於「抓柚子廚槍斃」或「抓ㄌㄌㄎ砲決」的功能。
>
>其中有五個可選參數：
>- 【enabled】：該伺服器是否啟用「抓柚子廚槍斃/抓我槍斃」的功能（True/False）
>  - 若為否，指令與頻道輸入「抓柚子廚槍斃」均不會觸發後續槍斃流程。
>  - 預設開啟。
>- 【owner_exempt】：伺服器擁有者是否受到槍斃豁免（True/False）
>  - 預設關閉。
>- 【allow_specify】：是否啟用指定用戶槍斃（True/False）
>  - 預設關閉。
>- 【max_incense_per_user】每人可上香次數。
>  - 數值限制為` -1 `~` 999 `之間的整數。
>     - 值為` -1 `時：每人可` 無限次 `上香。
>     - 值為` 0 `時：禁用上香按鈕。
>     - 值為` 1 `到` 999 `時：每人可上香次數依照輸入數字為主。
>  - 預設1次。
>- 【votesecond】投票時間
>  - 數值限制為` 5 `~` 3600 `之間的整數。
>  - 單位為秒
>  - 預設300秒

>[!Warning] 預設值說明
>坦若無設置，預設【enabled】為True(開啟)；【owner_exempt】為關閉；【allow_specify】為關閉；【max_incense_per_user】為1次；【votesecond】為600秒。
![image](https://hackmd.io/_uploads/Sk1yyKmOxe.png)
![image](https://hackmd.io/_uploads/BJxeJKXulg.png)

>[!Caution] 設置權限
>此指令**僅限伺服器內具有管理權的管理員才能設置**，
>如果您沒有管理權在指令清單頁面無法看見此指令。

# Q&A
:::warning
建議善用 **Ctrl+F** 搜尋關鍵字尋找問題答案。
:::


:::spoiler Q:任何人都能使用機器人嗎？
> **所有人都可以使用機器人。**
> :::danger
> **除了違反服務政策遭開發者停權者除外**。
> :::
:::

:::spoiler Q:使用機器人需要付費嗎？
> **不用，完全免費**。~~只是心血來潮弄的機器人~~
:::

:::spoiler Q:機器人是Yuzusoft（柚子社）官方機器人嗎？
> **不是！** 此機器人為民間柚子社愛好者創立的機器人，與柚子社官方無關。
> 但語錄相關之圖片智慧財產權為柚子社所有。
:::

:::spoiler Q:機器人有開啟哪個特權意圖(Intent)?
> **三大特權都沒有開啟**！包含伺服器成員、網關、訊息內容特權意圖。
:::

:::spoiler Q:如果機器人指令不能用怎麼辦？
> 先檢查機器人是否在線（機器人頭像顯示🟢而非🔘），若離線為機器人進行維護或突發性狀況離線，請參閱 [支援群](https://discord.gg/ZwPadtm5vr) 的 [開發公告](https://discord.com/channels/1378966069456408657/1379047784933756988) 頻道或詢問開發者。
> 
> 坦若機器人為在線🟢，則檢查該伺服器之頻道是否有開啟外部應用程式權限，如果您是該伺服器具備頻道管理（Manage Channels）或管理員權限（Administrator）者，可將「外部應用程式（Use External Apps）」及「使用應用程式（Use Application Commands）」權限開啟即可使用。
:::

:::spoiler Q:我被開發者停權了想申訴怎麼辦？
> 首先請加入 [支援群](https://discord.gg/ZwPadtm5vr) 後**私訊開發者**，申訴請填寫以下說明：
> 1. 申訴人
> 2. 申訴原因(理由)
> 3. 期望改正懲處

> 以上三者缺一不可，開發者不一定會即時回應但一定會回應，
> 請注意：申訴聲請將影響到開發者做出最終懲處（不予更改懲處、減緩停權時間或赦免）請認真填寫。
> 如果您提交了申訴瘋狂TAG開發者，一律駁回申訴+不得再次上訴。
:::

:::spoiler Q:新增的語錄會立即加入至指令清單裡嗎？
> 不會，原則上必須等到每周三凌晨0時重啟後生效，除非開發者立即重啟。
:::

:::spoiler Q:我想回報錯誤（漏洞），如何回報？
> 請加入 [支援群](https://discord.gg/ZwPadtm5vr) 後至 [#💥．錯誤回報](https://discord.com/channels/1378966069456408657/1379048940774621204) 論壇創立貼文回報，回報應包含以下資訊：

> 1. BUG說明（越詳細越好）
> 2. BUG如何重現（將重現步驟詳細說明，有利於開發者尋找問題根源）
> 3. 預期結果（預期原本應該的結果）
> 4. 附圖（盡量截圖問題）
:::

:::spoiler Q:我想為機器人提交語錄申請，如何提交申請？
> 請使用指令` /語錄申請 `申請，請詳閱 [申請語錄指令教學](https://hackmd.io/@yukicon0321/yuzusoft#%E7%94%B3%E8%AB%8B%E8%AA%9E%E9%8C%84%E6%8C%87%E4%BB%A4)
:::

:::spoiler Q:我也想成為` /關於機器人 `指令中銘謝清單的一員，如何申請？
> 銘謝清單僅銘謝協助語錄製作或回報錯誤的人，且要有至少三次以上的協助才可，達到條件後開發者會詢問您是否有意將自己納入銘謝清單裡，若您允許將加入至銘謝清單中一員，成為機器人協助員一份子。
:::

:::spoiler Q:機器人會公開代碼（開源）嗎？
> **不會**，因曾有人透過抄襲開發者自己文案進行牟利的缺德人士，故自那時起往後的開發者有關自製機器人均不會公開程式碼。
:::

:::spoiler Q:可以公開語錄圖檔嗎？
> **不會！** 請自行安裝遊戲後自行截圖。
:::

:::spoiler Q:抓柚子廚槍斃 / 抓ㄌㄌㄎ砲決功能能關閉嗎？
> **可以** ，請詳閱[抓柚子廚槍斃管理設定](https://hackmd.io/@yukicon0321/yuzusoft#抓柚子廚槍斃管理設定)。
:::

:::spoiler Q:抓柚子廚槍斃功能被槍斃者能獲得暫時性禁言嗎？
> **不行**　！此功能雖然以前有，但基於一些原因移除了。
:::

:::spoiler Q:機器人會自動異地備份檔案嗎？
> **會！** ，機器人檔案每五分鐘會將檔案異地備份（從2025/06/14起）。
:::

:::spoiler Q:我想依據隱私權政策申請消除機器人關於我的個人資訊怎麼辦?
>請加入 [支援群](https://discord.gg/ZwPadtm5vr) 後私訊開發者，表明您是要清除關於自己的個人資料，開發者看到後會依據您的意願進行刪除的動作，惟封禁清單無法清除任何資料。
:::


:::spoiler Q:語錄功能為原創功能嗎？
>**是的** ，因為發現《千恋＊万花》很多地方可以當語錄，然後想把它收集起來，所以誕生了。
>後面其他柚子社語錄也是配合我買了遊戲開始玩而新增的。
>包含查詢語錄、攻略建議、補丁教學、語錄申請等均為原創功能。
:::

:::spoiler Q:抓柚子廚槍斃 / 抓ㄌㄌㄎ砲決為原創功能嗎？
>**是的** ，一個是在玩蔚藍檔案想到的，一個是在廁所跟將軍交流想到的(?
:::

:::spoiler Q:婚姻系統為原創功能嗎？
>**不完全是，但部分特性為原創** ，一開始是在玩Yeecord機器龍時發現：
>欸靠怎不能養小三...?🤡
>然後就誕生了可以養小三的婚姻系統，並配合前面的抓柚子廚槍斃功能延伸下新增殉情系統。
:::

:::spoiler Q:定時重啟是什麼？
定時重啟是為了避免機器人在長時間開機下造成記憶體過度溢出導致崩潰，因此需要定時重啟來清理記憶體溢出空間。
:::

:::spoiler Q:機器人定時重啟是啥時？重啟前會有什麼機制？
> :::success 
> 機器人規定**每周三凌晨0時整定時自動重啟一次**。
> :::

每次定期重啟前，為了避免重啟造成的資料未被記錄或是按鈕壞掉，重啟前會階段式關閉部分指令運作，並於重啟完成後開放。



|  | 第一階段關閉 | 第二階段關閉 |
| -------- | -------- | -------- |
| 重啟前多久 | 重啟前15分鐘  | 重啟前15秒  |
| 具體時間 | 每周二23時45分0秒 ~ 每周三0時0分15秒 | 每周二23時59分45秒 ~ 每周三0時0分15秒 | 
| 關閉的指令 | /抓我槍斃、/抓柚子廚槍斃、/抓ㄌㄌㄎ砲決、/婚姻-求婚 | 所有指令 |




:::


:::spoiler Q:機器人版本號Alpha、Beta、正式版是什麼？
> Alpha代表早期開發版本，各種功能非常陽春
> Beta代表晚期開發版本，逐漸擴充各種功能與修復各種Bug，其中Beta版本可分七個階段：
> - Beta 1.3.2版本前：延續Alpha版本的柚子宇宙擴展更新部分
> - Beta 1.4.0 ~ Beta 1.7.18：後台與技術性更新Part I
> - Beta 1.8.0 ~ Beta 1.11.0：創新功能Part I
> - Beta 1.11.7 ~ Beta 1.12.2：後台與技術性更新Part II
> - Beta 1.13.0 ~ Beta 1.14.2：創新功能Part II
> - Beta 1.14.3 ~ Beta 1.14.13：後台與技術性更新Part III
> - Beta 1.15.0+：邁向正式版的前置更新
> 
> 正式版代表功能已成熟，且已修復大部分之錯誤，在此階段機器人會進行較少頻率的更新。

> 其中版本碼為 XXXX 1.Y.Z ，其中XXXX代表Alpha、Beta或正式版；Y代表該大版本更新編號；Z代表該附屬於大版本更新的次要更新編號，編號命名方式與Minecraft一樣。

> 備份版本碼是Beta 1.4.0新增的版本編號，主要是在版本碼後附註該次更新的備份檔案編號(下稱備份碼)，以利更新哪方面出問題迅速判斷調用哪個備份碼回退版本。
> 備份碼編號格式為YYWW-D-XXX 其中YY為21世紀該年分末兩碼(例如2025年，YY=25)；WW為該年第幾周(例如2025/07/31更新，屬於2025年第31周，WW=31)；D為該周星期幾(例如2025/07/31更新，屬於星期四，D=4。註:星期日D=7)；XXX為當天第幾次備份(1~約270)
:::
